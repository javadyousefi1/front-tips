Search & Highlight

Type in the search box to highlight all matching words in the text below.

The CSS Custom Highlight API is a powerful tool for styling text ranges without modifying the DOM. Unlike traditional approaches that wrap text in span elements, this API lets you create highlights programmatically using Range objects and register them with CSS. This makes it ideal for building search features, code editors, text annotation tools, and spell checkers. The API is supported in Chrome and Edge, and the performance is excellent even with thousands of highlighted ranges. Custom highlights can be layered — each highlight has its own name and its own CSS rule via the ::highlight() pseudo-element. 
const range = new Range();
range.setStart(textNode, startOffset);
range.setEnd(textNode, endOffset);

CSS.highlights.set('search-results', new Highlight(range));

/* In CSS */
::highlight(search-results) {
  background-color: #fff176;
  color: #000;
}

// Register multiple named highlights
CSS.highlights.set('keywords', new Highlight(...keywordRanges));
CSS.highlights.set('errors',   new Highlight(...errorRanges));
CSS.highlights.set('names',    new Highlight(...nameRanges));

/* Each has its own CSS rule */
::highlight(keywords) { background-color: #c8e6c9; }
::highlight(errors)   { text-decoration: underline wavy red; }
::highlight(names)    { background-color: #bbdefb; font-style: italic; }

document.addEventListener('selectionchange', () => {
  const selection = window.getSelection();
  if (selection.isCollapsed) return;

  const range = selection.getRangeAt(0).cloneRange();
  savedRanges.push(range);

  CSS.highlights.set('saved-selection', new Highlight(...savedRanges));
});
