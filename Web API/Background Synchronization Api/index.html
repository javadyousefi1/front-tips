📡 Demo 1: Basic Background Sync

Register a sync event and watch it fire automatically when connectivity returns

How Background Sync Works:

  • Register a sync event with a unique tag name
  • Browser fires the sync event in the Service Worker
  • If offline, it waits and fires as soon as you're back online
  • Automatically retries if the handler throws an error
  • Works even if the tab is closed!

Register a Sync Event

Type a tag name and register a one-time background sync:

Code Example:

// In your page JavaScript
const registration = await navigator.serviceWorker.ready;
await registration.sync.register('sync-messages');

// In your Service Worker (sync-sw.js)
self.addEventListener('sync', (event) => {
  if (event.tag === 'sync-messages') {
    event.waitUntil(sendPendingMessages());
  }
});

Offline Queue Pattern:

  • Register a background sync event
  • Service Worker picks up queued data and sends it
  • On success: item removed from queue
  • On failure: browser retries automatically

⏰ Demo 3: Periodic Background Sync

Register periodic sync to keep content fresh in the background

Periodic Background Sync:

  • Registers a recurring sync task with a minimum interval
  • Browser decides actual timing based on usage patterns
  • Requires the site to be installed as a PWA
  • Requires user permission (similar to notifications)
  • Fires the periodicsync event in the Service Worker

Periodic Background Sync requires:

  • The site must be installed as a PWA

🌐 Demo 5: Browser Support & Fallback

Check what's supported and implement graceful degradation

Graceful Degradation Strategy:

  • Check support before using any API
  • Always save data locally as a safety net
  • Fall back to online event listener
  • Or use direct fetch with retry logic
  • Never block the user interface

Your Browser's Support Status:

Fallback Demo: Online Event Listener

When Background Sync is unavailable, listen to the online event:

Fallback Log:

Fallback events will appear here...

Graceful Degradation Pattern:

async function syncData(data) {
  // Always save locally first!
  await saveToIndexedDB('pending', data);

  if ('serviceWorker' in navigator) {
    const reg = await navigator.serviceWorker.ready;

    if ('sync' in reg) {
      // Background Sync supported
      await reg.sync.register('sync-data');
      return { method: 'background-sync' };
    }
  }

  // Fallback: use online event listener
  if (navigator.onLine) {
    await sendToServer(data);
    return { method: 'direct' };
  }

  window.addEventListener('online', async () => {
    const pending = await getFromIndexedDB('pending');
    for (const item of pending) {
      await sendToServer(item);
    }
  }, { once: true });

  return { method: 'online-listener-fallback' };
}
