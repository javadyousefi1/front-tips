What is this API?

The Prioritized Task Scheduling API (scheduler.postTask()) allows you to schedule tasks with different priority levels:

  • user-blocking - Highest priority (critical UI updates)
  • user-visible - Default priority (most user interactions)
  • background - Lowest priority (analytics, prefetching)
