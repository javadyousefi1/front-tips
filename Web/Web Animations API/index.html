1. Basic Fade & Slide

Fundamentals

Simple opacity and transform animations

Animate Me!
element.animate([
  { opacity: 0, transform: 'translateY(-50px)' },
  { opacity: 1, transform: 'translateY(0px)' }
], {
  duration: 800,
  easing: 'ease-out',
  fill: 'forwards'
});

4. Playback Control

Advanced

Full control over animation playback

Status: idle Progress: 0% Speed: 1x 
const anim = element.animate(keyframes, options);

anim.play();              // Play/resume
anim.pause();             // Pause
anim.reverse();           // Reverse
anim.playbackRate = 2;    // Speed control
anim.finish();            // Jump to end

Key Concepts

  • element.animate(keyframes, options) - Core method
  • keyframes - Array of animation states
  • duration - Animation length in milliseconds
  • easing - Timing function (ease, linear, etc.)
  • iterations - Number of repeats (Infinity for loop)
  • fill - How to apply styles before/after animation
  • direction - normal, reverse, alternate

Animation Object Methods

  • play() - Start or resume
  • pause() - Pause animation
  • reverse() - Reverse direction
  • cancel() - Cancel and reset
  • finish() - Jump to end
  • playbackRate - Speed control
  • finished - Promise when complete
